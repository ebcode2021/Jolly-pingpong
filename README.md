## 5. 사용법

### 설치

```
$ git clone git@github.com:ebcode2021/Jolly-pingpong.git
$ cd Jolly-pingpong
```

### Backend

```
$ cd back-end
$ npm update
$ npm run start
```

### Frontend

```
$ cd front-end
$ npm update
$ npm run start
```

**서버 시작 후 사이트 접속** -> `http://localhost:5173`
